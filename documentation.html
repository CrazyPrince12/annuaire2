Documentation

CONGLOMERAT est un site statique HTML/CSS/JavaScript conçu pour Vercel. Les données sont séparées du code afin d’ajouter une ville ou une entreprise sans toucher aux scripts.

Ajouter une entreprise

Ouvrez le fichier JSON de la ville, ajoutez un objet dans entreprises, vérifiez l’absence de doublon nom + adresse, puis renseignez latitude, longitude, catégorie, hashtags et description FR/EN.

Ajouter une ville

Créez assets/data/entreprises-ville.json, puis référencez ce fichier dans assets/data/villes.json. La page entreprises génère automatiquement filtres, carte et compteurs.
